# 🚀 PanelQ – AI-Powered Interview Preparation Platform

PanelQ is a full-stack web application designed to help users practice interviews using AI. It provides real-time feedback, behavioral analysis, and technical evaluation to improve confidence and performance.

---

## 🌐 Live Preview

Frontend running at: `http://localhost:5173`

---

## 📌 Features

* 🤖 AI-powered mock interviews
* 📊 Real-time feedback and evaluation
* 🧠 Behavioral analysis
* 💻 Technical interview practice
* 🔐 User authentication (Login/Register)
* 📈 Performance tracking (optional future feature)

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/panelq.git
cd panelq
```

---

### 2️⃣ Setup Frontend

```bash
cd client
npm install
npm run dev
```

---

### 3️⃣ Setup Backend

```bash
cd server
npm install
nodemon index.js
```

---

### 4️⃣ Environment Variables

Create a `.env` file inside `server/` and add:

```
MONGO_URI=your_mongodb_connection_string
PORT=5000
JWT_SECRET=your_secret_key
```

---

## ▶️ How to Run

* Frontend: `npm run dev`
* Backend: `nodemon index.js`
